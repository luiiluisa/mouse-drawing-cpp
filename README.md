# mouse-drawing-cpp

A simple drawing application created in C++ using the `graphics.h` library.

The program allows the user to draw shapes on the screen with the mouse and change colors or shapes using the keyboard.

## Controls
Left Click – draw  
Right Click – erase  
UP Arrow – clear screen  
RIGHT Arrow – change shape (circle / square)  
LEFT Arrow – change color  
SPACE – exit
